------ Описание проекта
Основной файл - index.html. В нем подключены все блоки сайта, которые находятся в папке src/html/landing-blocks

------ Блок 1 (Загрузочная страница + страница с таймером)
Загрузочная страница - это элемент первого блока, который наложен поверх основного контента первого блока.
В десктопной версии проекта используется поблоковая прокрутка. В мобильной версии используется обычный скролл ввиду отсутствия места под контент.
На мобильной версии иначе работает анимация персонажа (появляется после скролла). В десктопной версии персонаж появляется сам с задержкой.
